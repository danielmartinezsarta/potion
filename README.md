# Potion ERP

Scalable Cloud based ERP for all size of companies.

## Dependencies

- Elixir 1.10.3
- Phoenix 1.5.6
- PosgreSQL 12

## Run instructions

- Clone this repo
- Make sure you have installed Elixir with mix, hex and the Phoenix generator
- Run `mix ecto.create`
- Run `mix phx.server`
- Go to http://localhost:4000/
